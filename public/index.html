Tweets From
Space!

SpaceTweets is a web application built in Portland, Oregon, United States. We developed and styled this application from the ground up using the Twitter and Google Maps APIs. The idea behind SpaceTweets is to give users the ability to look up tweets by hashtag or screen name and plot them by location. Enjoy!
